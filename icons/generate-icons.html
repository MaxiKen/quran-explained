📖 Al-Quran PWA Icon Generator

This page generates all required PWA icon sizes. Click "Download All" or download individually.

📋 Instructions

  1. Click "Download All" above to download each icon
  2. Place all downloaded PNG files into the icons/ folder
  3. The files must be named exactly as shown (e.g., icon-192x192.png)
  4. The manifest.json already references these filenames
  5. After adding the icons, the PWA will be installable on mobile devices

🎨 Customizing the Icon

  1. Edit the drawIcon() function in this file to change the design
  2. Regenerate and re-download the icons
  3. Or replace the PNG files with your own custom icons of the same sizes
